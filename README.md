<div align="center">
  <h1>📊 Student Score Display</h1>
  <p>A simple Python script that displays student names with their scores represented as star symbols.</p>
</div>

<hr />

<h2>🛠️ Technologies Used</h2>
<ul>
  <li><strong>Language:</strong> Python 3</li>
  <li><strong>Libraries:</strong> None (uses only built-in Python features)</li>
</ul>

<h2>✨ Features</h2>
<ul>
  <li>👨‍🎓 Displays a list of student names.</li>
  <li>⭐ Represents each student's score using <code>*</code> symbols.</li>
  <li>🧾 Prints output in a clean name-and-score format.</li>
  <li>⚡ Lightweight and easy to run.</li>
</ul>

<h2>🚀 Installation</h2>
<ol>
  <li>Make sure Python 3 is installed on your system.</li>
  <li>Save the file as <code>Student scores.py</code>.</li>
  <li>Open a terminal or command prompt.</li>
  <li>Run the script using the following command:</li>
</ol>

<pre><code>python "Student scores.py"</code></pre>

<h2>💻 How It Works</h2>
<p>
The program stores student names in one list and their scores in another list.
It then loops through both lists and prints each student's name followed by a number of stars equal to their score.
</p>

<h2>📥 Input & Output</h2>

<h3>Input</h3>
<p>No user input is required. The data is defined directly inside the script.</p>
